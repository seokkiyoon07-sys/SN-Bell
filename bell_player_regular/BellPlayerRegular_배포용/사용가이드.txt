# 평상시 종소리 프로그램 사용 가이드

## 🚀 빠른 시작

1. **BellPlayerRegular.exe** 파일을 더블클릭하여 실행
2. 사운드 폴더 경로를 설정 (기본: sounds 폴더)
3. "스케줄러 시작" 버튼 클릭

## 📁 파일 구성

- **BellPlayerRegular.exe**: 메인 프로그램 (15MB)
- **config.yaml**: 설정 파일
- **sounds/**: 종소리 파일 폴더 (01.mp3 ~ 20.mp3)
- **logs/**: 로그 파일 폴더
- **사용가이드.txt**: 이 파일

## ⚙️ 기본 설정

- **볼륨**: 0~1 사이 값
- **MCI 우선 사용**: Windows 내장 오디오 (권장)
- **테스트 모드**: 모든 종소리를 3초 간격으로 재생

## 🎮 사용법

1. **자동 재생**: "스케줄러 시작" → 고정된 시간에 자동 재생
2. **수동 재생**: 체크박스 선택 → "선택 재생"
3. **테스트**: "테스트 모드" 체크 후 시작

## 🔧 문제 해결

### 오디오 재생 안됨
1. "MCI 우선 사용" 체크박스 확인
2. 사운드 파일이 올바른 위치에 있는지 확인
3. 볼륨 설정 확인 (0~1 사이)

### 프로그램이 안 열림
1. Windows Defender에서 차단했을 수 있음
2. 관리자 권한으로 실행해보기
3. 바이러스 백신 프로그램에서 차단했을 수 있음
